AI/ML Engineer
We are seeking a highly skilled and motivated AI/ML Engineer to join our team. The ideal candidate will have hands-on experience in building and deploying machine learning and deep learning models to production. You will be responsible for the end-to-end machine learning lifecycle, from data preprocessing and model development to deployment and monitoring.

Responsibilities:

Design, develop, and deploy machine learning and deep learning models for various applications including Natural Language Processing (NLP) and Computer Vision.

Work with large-scale datasets, performing data cleaning, feature engineering, and data augmentation.

Implement and optimize ML algorithms using frameworks such as TensorFlow and PyTorch.

Build and maintain scalable ML pipelines (MLOps) on cloud platforms like AWS or GCP.

Collaborate with data scientists and software engineers to integrate models into production systems.

Required Skills & Qualifications:

Bachelor's or Master's degree in Computer Science, Data Science, or a related field.

Proficiency in Python programming.

Experience with ML libraries: Scikit-learn, Pandas, and NumPy.

Hands-on experience with deep learning frameworks: TensorFlow or PyTorch.

Strong understanding of MLOps practices, including model serving and monitoring.

Familiarity with containerization technologies (Docker) and orchestration (Kubernetes).

Preferred Skills:

Experience with cloud platforms (AWS, GCP).

Knowledge of SQL and distributed data processing frameworks like Spark.
